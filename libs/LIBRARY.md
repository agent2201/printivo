# 📚 Printivo Shared Library

This directory contains the core shared modules and documentation for the PrintServer ecosystem.

## 🛠 CORE TOOLSET (Dependencies)

These tools are installed in the environment and are ready for use in all services.

| Инструмент | Категория | Зачем это нам? |
| --- | --- | --- |
| **PyMuPDF** | Валидация | Проверка размеров и веса PDF файлов от клиентов. |
| **vtracer** | Векторизация | Мгновенное превращение фото в четкий вектор. |
| **rectpack** | Nesting | Умная упаковка наклеек на рулон без пустот. |
| **MinIO Client** | Storage | Связь Python-кода с вашим S3-хранилищем. |
| **ReportLab** | PDF Gen | Создание технических меток реза и штрих-кодов. |
| **Loguru** | Мониторинг | Профессиональное ведение логов всех ошибок сервера. |
| **Ghostscript** | PDF/CMYK | «Тяжелая» обработка и конвертация цвета для печати. |
